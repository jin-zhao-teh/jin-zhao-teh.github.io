Frequently Asked Questions

Find answers to common questions about IJPlus

Is IJPlus safe to use?

Yes, IJPlus is designed with security as a top priority. All scripts run in a sandboxed environment with proper permission controls to ensure your system remains secure.

What browsers are supported?

IJPlus currently supports Chrome, Firefox, Edge, and Safari. We're continuously working to expand browser compatibility.

Can I use my own scripts?

Absolutely! IJPlus allows you to write and execute your own custom JavaScript code. Our documentation provides guidance on best practices.

How do updates work?

IJPlus checks for updates automatically and can install them with a single click. You'll always have access to the latest features and security improvements.

Is there a mobile version?

Currently, IJPlus is designed for desktop browsers. We're exploring mobile solutions for future releases.
